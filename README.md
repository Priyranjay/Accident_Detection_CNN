# 🚗 Accident Detection using CNN

A deep learning project that detects whether a road accident has occurred by classifying images into **Accident** and **Non-Accident** using a Convolutional Neural Network (CNN) built with PyTorch.

---

## 📌 Overview

This project demonstrates the use of Computer Vision and Deep Learning for binary image classification. It can be used as a foundation for intelligent traffic monitoring and road safety applications.

---

## ✨ Features

- Binary image classification
- CNN model implemented in PyTorch
- Image preprocessing and prediction
- User-friendly Gradio interface
- Easy to extend for real-time detection

---

## 🛠 Tech Stack

- Python
- PyTorch
- OpenCV
- NumPy
- Gradio
